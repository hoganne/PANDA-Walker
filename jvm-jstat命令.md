# jstat命令

## 1.jstat命令简介

　　jstat命令可以查看堆内存各部分的使用量，以及加载类的数量。

　　命令的格式如下： jstat [-命令选项] [vmid] [间隔时间/毫秒] [查询次数]

## 2.使用详情列表

 **【以下的统计空间单位,未标明的 都是KB】**

**1>类加载统计**

命令：

```sh

```

结果：

![img](https://img2018.cnblogs.com/blog/978388/201906/978388-20190626144521902-1269472927.png)

解析：

```
Loaded:加载class的数量
Bytes：所占用空间大小
Unloaded：未加载数量
Bytes:未加载占用空间
Time：时间
```
